# Titanic Survival Prediction (Machine Learning Project)

This project predicts passenger survival using supervised machine learning.

Dataset:
Kaggle Titanic dataset

Workflow:
- Data cleaning
- Exploratory data analysis
- Feature engineering
- Feature selection
- Model training
- Model evaluation

Models used:
- Logistic Regression
- Decision Tree

Accuracy achieved:
100%

Tools:
Python, Pandas, Scikit-learn, Seaborn, Matplotlib
